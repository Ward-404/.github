# Ward 404

> *"Ward 404 does not appear in the hospital directory. It never did."*

---

## Where are we?

End of the hall. Last door. The ward that doesn't show up on any map,
any directory, or any official record. This is where the cases that
didn't fit anywhere else ended up. Turns out — that's where the work
gets done.

We were referred here by reality. Rejected by convention.
We never left.

---

## What happens inside

We build things. Tools, experiments, scripts, systems — whatever
needs to exist that doesn't yet. Nothing too mainstream. Nothing
too expected. If it's already been done a thousand times, it's
probably not coming from this ward.

- Some repos will be useful
- Some will be strange
- All of them will be intentional

---

## Who's in here

A small collective of high-functioning outliers. Diagnosis varies.
Output does not.

Contributions are by invitation. If you found this place,
you probably belong here.

---
